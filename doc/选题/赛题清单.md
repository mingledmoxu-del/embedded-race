## 2. 各赛道题目清单

### 2.1 ST 赛道

1. 嵌入式人工智能
2. 智能精密数字电源与测量仪器
3. 车规 MCU 专题-智能飞行汽车
4. 工业 4.0 / 电机控制 / 机器人
5. 智能可穿戴设备
6. IoT
7. 自主选题

推荐关注：
- 选题二：智能精密数字电源与测量仪器
- 选题四：工业 4.0 / 电机控制 / 机器人
- 选题五：智能可穿戴设备

原因：
- 技术壁垒高，做得好容易和普通 AI 演示类作品拉开差距。
- 更容易体现控制、采样、实时性、可靠性和工程能力。

### 2.2 ESP 赛道

1. 嵌入式边缘 AI 应用
2. 高性能图形人机交互系统
3. 电机驱动与运动控制
4. 无线协议与网络应用
5. 智能交互驱动的 AIoT 应用系统
6. 自主命题

推荐关注：
- 选题三：电机驱动与运动控制
- 选题五：智能交互驱动的 AIoT 应用系统
- 选题二：高性能图形人机交互系统

原因：
- 平台生态成熟，容易做出完整样机。
- 但聊天机器人、桌宠、语音助手类作品可能扎堆，必须做差异化。

### 2.3 NXP 赛道

#### 开放式自主选题

1. 人工智能、机器学习方向
2. 工业和自动化方向
3. IoT、智物联网、智慧家居方向
4. 自主选题

#### 挑战创新选题

1. 无人机地面站视觉识别终端
2. 机械臂危险区域入侵检测
3. 智能家居 AI 监护与交互系统
4. 网络摄像头
5. 仓储物流 AI 智能分拣系统
6. 工业监测系统
7. 视觉 + EtherCAT 的智能机器人
8. 轻量工业 HMI

推荐重点关注：
- 挑战创新选题二：机械臂危险区域入侵检测
- 挑战创新选题五：仓储物流 AI 智能分拣系统
- 挑战创新选题七：视觉 + EtherCAT 的智能机器人
- 挑战创新选题三：智能家居 AI 监护与交互系统

原因：
- 题目本身就是围绕真实工业或民生场景设计。
- 很容易形成“感知-分析-决策-联动-回放”的完整系统闭环。
- 评委更容易看到产业落地价值。

### 2.4 RK 赛道

1. 边缘人工智能应用
2. 端侧 AI 视觉应用
3. 智慧工业应用
4. 自主选题

推荐关注：
- 选题二：端侧 AI 视觉应用
- 选题一：边缘人工智能应用
- 选题三：智慧工业应用

原因：
- 算力强，适合做视觉、多模态和本地推理。
- 但必须加入系统管理、复杂场景鲁棒性和工程化设计，否则容易像普通视觉 Demo。

## 3. 冲奖优先级建议

我的建议排序：

1. NXP
2. ST
3. RK
4. ESP

排序依据：
- NXP：工程场景完整，命题清晰，容易做出高价值闭环。
- ST：技术深度强，适合做硬核工程作品。
- RK：展示效果强，但作品同质化风险较高。
- ESP：生态成熟，但热门题型容易撞车。

## 4. 最值得优先考虑的题目

### 4.1 NXP 挑战创新选题二：机械臂危险区域入侵检测

推荐指数：`极高`

优势：
- 工业安全场景明确
- 不强依赖真实机械臂
- AI 检测和控制联动都能体现
- 答辩展示直观
- 容易做出完整闭环

### 4.2 ST 选题二：智能精密数字电源与测量仪器

推荐指数：`很高`

优势：
- 作品稀缺，技术辨识度高
- 容易体现控制、采样、DSP、保护机制
- 做好后很有专业感

### 4.3 NXP 挑战创新选题七：视觉 + EtherCAT 的智能机器人

推荐指数：`很高`

优势：
- 视觉、工业通信、实时控制三者结合
- 非常符合高水平竞赛项目特征

注意：
- 开发难度和集成风险都较高

## 5. 可重点发散的赛题灵感

1. 机械臂动态安全域防护终端
   方向：NXP 挑战创新选题二
   卖点：危险区动态变化、分级告警、事件回放、GPIO 或网络联动急停

2. 仓储物流 OCR + 条码异常分拣终端
   方向：NXP 挑战创新选题五
   卖点：反光、堆叠、遮挡场景识别，支持本地缓存和云端同步

3. 可编程精密电源 + 波形分析一体机
   方向：ST 选题二
   卖点：电源输出、谐波分析、曲线显示、异常预测、保护机制

4. 工业设备预测性维护节点
   方向：ST 选题一或选四
   卖点：振动、电流、温度多参数采集，边缘异常检测，本地告警和无线回传

5. 低空巡检视觉分析终端
   方向：RK 选题二 或 NXP 挑战创新选题一
   卖点：目标检测、轨迹跟踪、越界告警、弱网下稳定传输

6. 多模态居家监护与交互终端
   方向：NXP 挑战创新选题三 或 ESP 选题五
   卖点：跌倒检测、身份识别、触摸交互、远程通知、家居联动

## 6. 当前建议

如果目标是优先拿奖，我建议首先考虑以下两个方向之一：

1. `NXP 挑战创新选题二：机械臂危险区域入侵检测`
2. `ST 选题二：智能精密数字电源与测量仪器`

前者更容易做出“评委一眼能看懂”的高分系统。

后者更容易做出“技术含量很强”的硬核作品。
